# Future Platform Roadmap

This document preserves the longer-term productization direction for Agent Contracts after the benchmark is thesis-ready.

## Phase 1: Infrastructure and Scaling

- Dynamic contract schema via plugin-style rule kinds.
- Versioned contract registry for distributed agents.
- Enhanced benchmark materialization for multi-agent synthetic history generation.

## Phase 2: Live Runtime Enforcement

- LangGraph or LangChain middleware for validation callbacks and state guards.
- CrewAI and AutoGen interceptors for contract-aware agent messaging.
- Permissive versus strict enforcement policies.
- Operational circuit breakers for repeated high-severity violations.

## Phase 3: Ecosystem Integration

- MCP-compatible Governor server for IDE and context-client audits.
- CI integration for contract checks on pull requests.
- Alignment with emerging observability conventions for agent traces.

## Phase 4: Advanced Governance

- Semantic validation for documentation-capability alignment.
- Contract mining from historical trajectories.
- Coordination contracts for multi-agent workflows and shared resources.
